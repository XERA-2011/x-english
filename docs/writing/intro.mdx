# ✍️ 英语写作

> 写作是语言输出的重要形式，是将语法和词汇知识转化为实际应用的过程。

## 学习路线

### ✏️ 写作基础 <DifficultyBadge level="A2" />

- **基本句型**：掌握五种基本句型的灵活运用
- **段落结构**：主题句 + 支持句 + 结尾句
- **标点符号**：正确使用英文标点

### 📄 文体写作 <DifficultyBadge level="B1" />

- **日常邮件**：问候、请求、感谢、道歉
- **议论文**：观点表达、论证、总结
- **报告**：结构化表达与数据描述

### 🎯 高级写作 <DifficultyBadge level="B2" />

- **衔接与连贯**：过渡词、指代、逻辑关系
- **写作风格**：正式与非正式、学术与日常
- **修辞手法**：比喻、排比、反问

---

:::tip 写作提升秘诀
1. **模仿优秀范文**：分析好文章的结构和用词
2. **先写后改**：不要追求一次写完美，先写出来再修改
3. **注重逻辑**：使用连接词让文章层次分明
4. **积累好词好句**：建立自己的写作素材库
:::

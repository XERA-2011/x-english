# 🎤 英语发音

> 准确的发音是有效沟通的前提。学好发音，让你的英语更地道、更自信。

## 学习路径

### 🔤 音标系统 <DifficultyBadge level="A1" />

国际音标 (IPA) 是学习发音的基础工具：

| 类别 | 数量 | 内容 |
|------|------|------|
| 元音 (Vowels) | 20 个 | 单元音 12 个 + 双元音 8 个 |
| 辅音 (Consonants) | 28 个 | 清辅音 + 浊辅音 + 其他 |

### 🎵 发音规则 <DifficultyBadge level="B1" />

- **连读 (Linking)**：辅音 + 元音的自然衔接
- **重音 (Stress)**：单词重音与句子重音
- **语调 (Intonation)**：升调、降调与含义变化
- **弱读 (Reduction)**：功能词在句中的弱化

### 🗣️ 发音练习 <DifficultyBadge level="A2" />

- **绕口令 (Tongue Twisters)**：训练口腔肌肉灵活度
- **最小对立体 (Minimal Pairs)**：区分易混淆的发音

---

:::tip 发音练习建议
1. **多听多模仿**：跟着音频反复练习
2. **录音对比**：录下自己的发音与原声对比
3. **注意口型**：元音发音时特别注意嘴型变化
4. **从单词到句子**：先练准单个音，再练连读和语调
:::

# 📚 英语阅读

> 阅读是扩展词汇量、提升语感最有效的方式之一。

## 分级阅读

| 级别 | 词汇要求 | 适合内容 |
|------|---------|---------|
| <DifficultyBadge level="A1" /> | 500 词以内 | 简单对话、短句、日常标语 |
| <DifficultyBadge level="A2" /> | 1000 词以内 | 简短故事、基础新闻 |
| <DifficultyBadge level="B1" /> | 2000 词以内 | 一般新闻、简单文章 |
| <DifficultyBadge level="B2" /> | 5000 词以内 | 杂志文章、小说节选 |
| <DifficultyBadge level="C1" /> | 8000+ 词 | 学术论文、专业文献 |

## 阅读策略

- **略读 (Skimming)**：快速浏览获取大意
- **扫读 (Scanning)**：搜索特定信息
- **精读 (Intensive Reading)**：逐字逐句深入理解
- **批判性阅读 (Critical Reading)**：分析论点和逻辑

---

:::tip 阅读习惯养成
1. 每天至少阅读 **15 分钟**英文材料
2. 先**通读全文**，不要遇到生词就查
3. 通过**上下文推测**生词含义
4. 读后**用自己的话总结**主要内容
:::

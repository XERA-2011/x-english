# 📖 英语语法

> 语法是英语的骨架。掌握语法，才能真正理解和运用英语。

## 学习路径

本模块将带你系统学习英语语法，从最基础的时态开始，逐步深入到复杂句式。

### 🟢 入门阶段 <DifficultyBadge level="A1" /> <DifficultyBadge level="A2" />

- **时态基础**：一般现在时、一般过去时、一般将来时
- **基本句型**：五种基本句型结构
- **词性入门**：名词、动词、形容词

### 🟡 提升阶段 <DifficultyBadge level="B1" /> <DifficultyBadge level="B2" />

- **进阶时态**：完成时、进行时、完成进行时
- **从句**：定语从句、状语从句、名词性从句
- **被动语态**：各时态的被动形式

### 🔴 高级阶段 <DifficultyBadge level="C1" />

- **虚拟语气**：与事实相反的假设
- **非谓语动词**：不定式、动名词、分词
- **特殊句式**：倒装、强调、省略

## 学习建议

:::tip 高效学习法
1. **先理解概念**，再看例句
2. **做练习题**巩固理解
3. **在阅读和写作中应用**语法知识
4. 不必追求完美，**先掌握核心规则**
:::

# 非谓语动词 Non-finite Verbs <DifficultyBadge level="B2" />

> 非谓语动词指不作谓语的动词形式，包括**不定式 (to do)**、**动名词 (-ing)** 和**分词 (-ing / -ed)**。

## 一、不定式 to do

### 作主语

<ExampleSentence
  en="To learn a new language takes time and effort."
  cn="学一门新语言需要时间和努力。"
  highlights={["To learn"]}
  note="不定式作主语。口语中更常用 It takes time to learn..."
/>

### 作宾语

<ExampleSentence
  en="She decided to study abroad."
  cn="她决定出国留学。"
  highlights={["to study"]}
  note="decide/want/hope/plan/agree 等后接不定式"
/>

### 作目的状语

<ExampleSentence
  en="He went to the library to borrow some books."
  cn="他去图书馆借了一些书。"
  highlights={["to borrow"]}
  note="不定式表示目的"
/>

### 常见接不定式的动词

| 动词 | 含义 |
|------|------|
| want / wish / hope | 想要 / 希望 |
| decide / choose | 决定 / 选择 |
| plan / agree / refuse | 计划 / 同意 / 拒绝 |
| manage / afford / promise | 设法 / 负担得起 / 承诺 |

## 二、动名词 -ing

### 作主语

<ExampleSentence
  en="Swimming is good exercise."
  cn="游泳是很好的运动。"
  highlights={["Swimming"]}
  note="动名词作主语，谓语用单数"
/>

### 作宾语

<ExampleSentence
  en="She enjoys reading novels."
  cn="她喜欢读小说。"
  highlights={["reading"]}
  note="enjoy/finish/avoid/mind 等后接动名词"
/>

### 常见接动名词的动词

| 动词 | 含义 |
|------|------|
| enjoy / finish / avoid | 喜欢 / 完成 / 避免 |
| mind / suggest / consider | 介意 / 建议 / 考虑 |
| practice / keep / give up | 练习 / 持续 / 放弃 |
| admit / deny / appreciate | 承认 / 否认 / 感激 |

## 三、接不定式和动名词含义不同的动词

| 动词 | + to do | + -ing |
|------|---------|--------|
| **remember** | 记得要做（未做） | 记得做过（已做） |
| **forget** | 忘记要做（未做） | 忘记做过（已做） |
| **stop** | 停下来去做（另一件事） | 停止做（当前的事） |
| **try** | 尽力做 | 尝试做 |

<ExampleSentence
  en="I remember locking the door. (I locked it.)"
  cn="我记得锁了门。（已经锁了）"
  highlights={["remember locking"]}
  note="remember doing = 记得做过某事（已发生）"
/>

<ExampleSentence
  en="Remember to lock the door. (You haven't done it yet.)"
  cn="记得锁门。（还没锁）"
  highlights={["Remember to lock"]}
  note="remember to do = 记得去做某事（未发生）"
/>

## 四、分词

### 现在分词 (-ing)：主动 / 正在进行

<ExampleSentence
  en="The sleeping baby looks very cute."
  cn="睡着的宝宝看起来很可爱。"
  highlights={["sleeping"]}
  note="现在分词作定语，表示主动：baby 在睡觉"
/>

### 过去分词 (-ed)：被动 / 已完成

<ExampleSentence
  en="The broken window needs to be repaired."
  cn="那扇破损的窗户需要修理。"
  highlights={["broken"]}
  note="过去分词作定语，表示被动：window 被打破"
/>

### 分词作状语

<ExampleSentence
  en="Walking along the street, I met an old friend."
  cn="走在街上，我遇到了一位老朋友。"
  highlights={["Walking along the street"]}
  note="现在分词作伴随状语，主语是 I（我在走路）"
/>

## 五、实战练习

<Quiz
  type="multiple-choice"
  question="She avoided _____ the difficult question."
  options={["answer", "to answer", "answering", "answered"]}
  answer="answering"
  explanation="avoid 后只能接动名词 (-ing)。avoid answering = 避免回答。"
/>

<Quiz
  type="multiple-choice"
  question="I remember _____ this movie before. I've seen it."
  options={["to watch", "watching", "watch", "watched"]}
  answer="watching"
  explanation="remember doing = 记得做过某事（已发生）。这里说'我之前看过'，用 remember watching。"
/>

<Quiz
  type="fill-blank"
  question="He decided _____ (go) abroad for further study."
  answer="to go"
  explanation="decide 后接不定式 to do。decide to go = 决定去。"
/>

<Quiz
  type="fill-blank"
  question="The _____ (excite) children ran to the playground."
  answer="excited"
  explanation="修饰人的感受用过去分词 excited（感到兴奋的），不是 exciting（令人兴奋的）。"
/>

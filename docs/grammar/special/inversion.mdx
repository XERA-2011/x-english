# 倒装句 Inversion <DifficultyBadge level="B2" />

> 倒装是将谓语（或助动词）放到主语前面的语序。四级考试中常以选填/改错形式考查。

## 一、全部倒装

**整个谓语置于主语之前**，常见于以下场景：

### 1. 地点/方位副词开头

<ExampleSentence
  en="Here comes the bus!"
  cn="公交车来了！"
  highlights={["Here comes"]}
  note="here/there + 完整谓语 + 名词主语。注意：主语为代词时不倒装（Here it comes.）"
/>

<ExampleSentence
  en="On the wall hangs a beautiful painting."
  cn="墙上挂着一幅美丽的画。"
  highlights={["On the wall hangs"]}
  note="介词短语提前 + 谓语 + 主语"
/>

## 二、部分倒装

**助动词/情态动词置于主语之前**（谓语的实义部分不变），更常考。

### 1. 否定词开头

<ExampleSentence
  en="Never have I seen such a beautiful sunset."
  cn="我从未见过如此美丽的日落。"
  highlights={["Never have I"]}
  note="否定副词 never 提前，助动词 have 放到主语 I 前面"
/>

<ExampleSentence
  en="Not only does he speak English, but also he speaks French."
  cn="他不仅会说英语，还会说法语。"
  highlights={["Not only does he"]}
  note="Not only 开头，前半句倒装，but also 后的分句不倒装"
/>

### 常见引起倒装的否定词

| 词/短语 | 含义 |
|---------|------|
| never | 从不 |
| hardly / scarcely | 几乎不 |
| seldom / rarely | 很少 |
| not only...but also | 不仅…而且 |
| not until | 直到…才 |
| by no means | 绝不 |
| in no way | 决不 |

### 2. only + 状语开头

<ExampleSentence
  en="Only by working hard can you achieve success."
  cn="只有努力工作你才能获得成功。"
  highlights={["Only by working hard can"]}
  note="only + 介词短语提前，主句部分倒装"
/>

### 3. so/such...that 中 so/such 提前

<ExampleSentence
  en="So beautiful was the scenery that we decided to stay longer."
  cn="景色如此美丽，以至于我们决定多待一会儿。"
  highlights={["So beautiful was"]}
  note="so + adj. 提到句首，主句倒装"
/>

### 4. as 引导的让步从句

<ExampleSentence
  en="Young as he is, he has rich experience."
  cn="尽管他年轻，但他经验丰富。"
  highlights={["Young as he is"]}
  note="adj./adv./n. + as + 主语 + 谓语 = 虽然…"
/>

## 三、实战练习

<Quiz
  type="multiple-choice"
  question="Never _____ such a wonderful performance."
  options={["I have seen", "have I seen", "I saw", "did I saw"]}
  answer="have I seen"
  explanation="否定词 Never 开头，部分倒装：助动词 have 置于主语 I 前面。"
/>

<Quiz
  type="multiple-choice"
  question="Only when he arrived _____ the truth."
  options={["he knew", "did he know", "he did know", "knew he"]}
  answer="did he know"
  explanation="only + 时间状语从句开头，主句部分倒装：did he know。"
/>

<Quiz
  type="fill-blank"
  question="Not only _____ he speak English, but he also speaks Japanese."
  answer="does"
  explanation="Not only 开头，前半句部分倒装：Not only does he speak...。"
/>

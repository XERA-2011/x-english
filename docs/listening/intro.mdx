# 🎧 英语听力

> 听力是语言输入的第一道关卡。能听懂，才能真正融入英语环境。

## 分级听力

| 级别 | 语速 | 适合材料 |
|------|------|---------|
| <DifficultyBadge level="A1" /> | 慢速 (~100 wpm) | 简单指令、基础对话 |
| <DifficultyBadge level="A2" /> | 较慢 (~120 wpm) | 日常对话、简单故事 |
| <DifficultyBadge level="B1" /> | 正常偏慢 (~140 wpm) | 新闻播报（慢速版） |
| <DifficultyBadge level="B2" /> | 正常 (~160 wpm) | 播客、电影对白 |
| <DifficultyBadge level="C1" /> | 快速 (~180+ wpm) | 学术讲座、辩论 |

## 听力策略

- **笔记法 (Note-taking)**：边听边记关键词
- **预测法 (Prediction)**：根据上下文预测内容
- **精听 (Intensive Listening)**：逐句听写
- **泛听 (Extensive Listening)**：大量接触，培养语感

## 推荐资源

| 资源 | 级别 | 特点 |
|------|------|------|
| VOA Learning English | A1-A2 | 慢速英语，配有文本 |
| BBC Learning English | A2-B1 | 短视频，话题丰富 |
| TED Talks | B1-C1 | 各领域专家演讲 |
| NPR Podcasts | B2-C1 | 新闻与深度报道 |

---

:::tip 听力突破技巧
1. **精听与泛听结合**：每天精听 15 分钟 + 泛听 30 分钟
2. **先听后看**：先不看文本听两遍，再对照文本查漏补缺
3. **跟读练习 (Shadowing)**：紧跟音频朗读，提升语感
4. **注意连读弱读**：这是听力最大的障碍之一
:::

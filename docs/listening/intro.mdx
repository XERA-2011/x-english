# 🎧 英语听力

> 听力是语言输入的第一道关。听得懂，才能说得出。

## 听力分级

| 级别 | 典型内容 | 核心能力 |
|------|---------|---------|
| <DifficultyBadge level="A2" /> | 简单指令、日常问候 | 识别关键词 |
| <DifficultyBadge level="B1" /> | 短对话、生活场景 | 捕捉主旨与细节 |
| <DifficultyBadge level="B2" /> | 长对话、新闻采访 | 推理判断、信息整合 |
| <DifficultyBadge level="C1" /> | 段落听写、学术讲座 | 精准听辨与拼写 |

## 听力策略

- **预读选项 (Preview)**：播放前快速浏览题目与选项，带着问题去听
- **关键词捕捉 (Key Words)**：抓住时间、地点、人物、数字等核心信息
- **逻辑推理 (Inference)**：结合语境推断言外之意，注意转折词 *but / however*
- **笔记速记 (Note-taking)**：长对话中用符号快速记录要点

## 开始练习

- 💬 **[短对话理解](./short-conversations)** <DifficultyBadge level="B1" /> — 日常场景短对话，快速捕捉关键信息
- 🗣️ **[长对话理解](./long-conversations)** <DifficultyBadge level="B2" /> — 学术、职场、新闻等长篇对话，训练推理与细节
- ✍️ **[短文听写](./dictation)** <DifficultyBadge level="C1" /> — 整段听写，提升精听能力与拼写准确度

---

:::tip 听力习惯养成
1. 每天坚持 **15–30 分钟**精听训练
2. 先**盲听**一遍，再对照原文复听
3. 重点关注**连读、弱读、语调变化**
4. 错题回顾，总结**常见干扰项规律**
:::

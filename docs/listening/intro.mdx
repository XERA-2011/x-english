# 🎧 英语听力实战

听力是语言输入的重要途径。本模块为您提供高密度的听力实战训练，请选择以下模块开始练习：

## 练习模块

- **[短对话理解](./short-conversations)** <DifficultyBadge level="B1" />
  日常场景的简短对话（2-4句），快速捕捉关键信息。
  
- **[长对话理解](./long-conversations)** <DifficultyBadge level="B2" />
  学术讨论、工作面试、新闻采访等较长篇幅对话，训练推理与细节捕捉。

- **[短文听写](./dictation)** <DifficultyBadge level="C1" />
  完整段落的听写训练，提升听力精准度与拼写能力。

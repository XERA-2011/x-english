# 📝 英语词汇

> 词汇是语言的基石。词汇量的大小直接影响你的听说读写能力。

## 学习路径

### 📊 按频率分级

| 词汇量 | 级别 | 能力描述 |
|--------|------|----------|
| 500 词 | <DifficultyBadge level="A1" /> | 基本生存用语，简单日常交流 |
| 1000 词 | <DifficultyBadge level="A2" /> | 日常基本交流，理解简单文章 |
| 2000 词 | <DifficultyBadge level="B1" /> | 较流利的日常交流，阅读一般文章 |
| 5000 词 | <DifficultyBadge level="B2" /> | 自如交流，阅读大部分英文内容 |
| 8000+ 词 | <DifficultyBadge level="C1" /> | 接近母语水平，理解专业内容 |

### 📚 模块内容

- **核心词汇**：按使用频率排列的高频词
- **场景词汇**：日常生活、商务、科技、旅行等
- **构词法**：前缀、后缀、词根——一次学会一类词
- **易混淆词**：affect/effect、lie/lay 等常见混淆词辨析

---

:::tip 词汇记忆法
1. **语境记忆**：将词汇放在句子中理解，而非孤立背诵
2. **词根联想**：学会常见词根词缀，举一反三
3. **间隔重复**：利用翻转词卡定期复习
4. **主动使用**：在写作和口语中有意识地使用新学词汇
:::

# 构词法与词根 <DifficultyBadge level="B2" />

> 授人以鱼不如授人以渔。英语单词大多由"前缀 + 词根 + 后缀"组成。掌握高频词根词缀，能让你在遇到生词时"猜"出它的意思，实现词汇量指数级增长。

## 一、高频前缀 (Prefixes)

前缀通常用来**改变单词的词义**，例如表示否定、方向、时间或程度。

后缀通常用来**改变单词的词性**，例如把动词变成名词，把名词变成形容词。

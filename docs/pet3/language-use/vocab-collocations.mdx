# PET3 完形填空常考搭配与词卡自测 <DifficultyBadge level="B1" />

> 完形填空中有超过 30% 的题目直接考查**固定搭配 (Fixed Collocations)**。这类题不需要过度进行语义逻辑推导，只要能识别出前后词汇的共现关系，即可在 5 秒钟内瞬间锁定正确答案。

---

## 🏛️ 一、高频必背固定搭配分类汇总

为了帮助你高效记忆，我们将 PETS-3 完形填空中出镜率最高的搭配整理为三大核心阵营：

### 1. 🎬 核心“动词 + 名词”搭配
* `make a decision` (做出决定) / `make progress` (取得进步) / `make a difference` (产生重大影响)
* `take responsibility for` (对……负责) / `take measures` (采取措施) / `take part in` (参加)
* `pay attention to` (注意) / `pay a visit to` (访问) / `pay for` (为……付款)
* `have access to` (有权进入/使用) / `have trouble doing` (做某事有困难) / `have an effect on` (对……产生影响)

### 2. 🎨 核心“形容词 + 介词”搭配
* `be responsible for...` (对……负责)
* `be familiar with...` (对……很熟悉)
* `be different from...` (与……不同)
* `be satisfied with...` (对……感到满意)
* `be interested in...` (对……感兴趣)

### 3. 🧩 核心名词/副词短语搭配
* `in terms of...` (在……方面 / 就……而言)
* `in addition to...` (除……之外还)
* `as a result` (结果) / `as a matter of fact` (事实上)
* `by contrast` (相比之下) / `at the same time` (同时)

## 💡 考场秒杀动作提示：
当你在完形填空空格前后看到 `access` 时，视线立刻去选项中寻找 `have` 或后面的介词 `to`；当你在空格后看到 `responsibility` 时，视线立刻寻找动词 `take` 或介词 `for`。这种**“首尾相连”的快判技巧**，能为你节省大量的阅读时间！

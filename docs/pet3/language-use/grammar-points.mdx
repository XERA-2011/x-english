# PET3 完形填空核心语法快判 <DifficultyBadge level="B1" />

> 在完形填空中，语法考题占比近 40%。解决语法空格的精髓在于**“辨识句子主干，寻找语法锚点”**。本节为你精梳最常考的四大语法考点，并以可视化的形式解构句子结构。

---

## 🏛️ 一、四大核心语法高频考点

### 1. 非谓语动词 (Non-finite Verbs) 🌟 最核心
* **考点形式**：给出同一个动词的 `to do` (不定式), `doing` (现在分词/动名词), `done` (过去分词) 等形式混选。
* **快速判词动作**：
  1. 第一步：观察空格所在句**是否已经存在谓语动词**。
     - 若**不存在** ➡️ 填**谓语动词**（注意考虑时态和单复数）。
     - 若**已存在** ➡️ 填**非谓语动词**。
  2. 第二步：分析非谓语动词与修饰名词的**主动/被动关系**。
     - 主动/进行 ➡️ 选择 `doing`。
     - 被动/完成 ➡️ 选择 `done`。
     - 目的/将来 ➡️ 选择 `to do`。

---

### 🔍 句子成分图解：非谓语后置定语

在阅读和完形填空中，非谓语动词常作为后置定语来修饰名词。让我们通过下方图解理清其在句中的成分角色：

    { text: "a part-time job application", role: "宾语 (Object)", color: "#EF4444" }

/>

> **快判解析**：句中已经拥有了谓语动词 `is writing`，因此修饰主语 `The student` 的成分必须使用非谓语形式。因为学生与“站立(stand)”是主动关系，所以选用现在分词短语 `standing by the window` 作后置定语。

---

### 2. 时态一致性 (Tense Coordination)
* **考点形式**：一般过去时、现在完成时、一般现在时等形式的辨析。
* **快速判词动作**：
  - **寻找时间锚点**：寻找句中的时间状语。
    - `yesterday / last year / in 1998` ➡️ 强锁**一般过去时**。
    - `since / for + 时间段 / so far / up to now` ➡️ 强锁**现在完成时**。
  - **上下文时态回看**：如果空格所在句没有时间状语，直接回看前后两句的主动词时态。完形填空通常全篇叙事，时态具有极强的惯性（如全篇过去时叙事，空格句无特殊情况一般也是过去时）。

---

### 3. 主谓一致性 (Subject-Verb Agreement)
* **考点形式**：单数谓语 (is / has / does) 与复数谓语 (are / have / do) 的混淆。
* **快速判词动作**：
  - **剥离干扰，锁定核心主语**：考官极其喜欢在主语后面安插一长串的修饰成分，干扰考生的单复数判断。
  - **破解**：无视介词短语（如 `of / with / as well as` 引导的成分）或后置定语，直接抓起最核心的名词跟谓语做匹配。

#### 🔍 句子成分图解：主谓一致干扰

<GrammarDiagram
  sentence="The list of high-frequency vocabulary words has been updated by the teacher."
  parts={[
    { text: "The list", role: "核心主语 (Singular Subject)", color: "#18E299" },
    { text: "of high-frequency vocabulary words", role: "介词短语干扰 (Modifier)", color: "#8B5CF6" },
    { text: "has been updated", role: "单数谓语 (Singular Verb)", color: "#EAB308" }
  ]}
/>

> **快判解析**：虽然空格前紧挨着复数名词 `words`，但本句的真正主语是单数名词 `The list`。因此谓语动词必须使用单数形式的 `has been updated`，绝对不能用复数 `have`。

---

### 4. 定语从句引导词 (Relative Pronouns)
* **考点形式**：`which`、`that`、`who`、`whom`、`whose` 等代词的混选。
* **快速判词动作**：
  1. 找准**先行词**：
     - 先行词是**人** ➡️ 考虑 `who / whom / whose`。
     - 先行词是**物** ➡️ 考虑 `which / that / whose`。
  2. 看空格在从句中**充当的成分**：
     - 缺主语/宾语 ➡️ 选择 `which / that / who`。
     - 缺定语（表示“……的”） ➡️ 选择 `whose`（如：*whose father is a doctor*）。
     - 句子完整，缺时间/地点状语 ➡️ 选择 `when / where`（等于 *prep. + which*）。

# PET3 阅读理解 <DifficultyBadge level="B1" />

> 阅读提分的关键不是“读得慢而全”，而是“按题型精准定位 + 快速排除干扰项”。

## 一、先明确阅读在考什么

PETS3 阅读核心考四种能力：

- 获取事实细节（人、地、时、因果）
- 理解词句含义（词义、指代、句间关系）
- 概括主旨要义（段落中心、全文主题）
- 进行合理推断（态度、意图、结论）

## 二、6步解题法（考场直接用）

1. `先看题`：浏览题干，圈关键词（人名、时间、专有名词）
2. `后读文`：按段落快速扫读，标主题句与转折词
3. `先易后难`：先做细节题和词义题，后做主旨与推断题
4. `回文定位`：以题干关键词回到原文找“同义替换”
5. `排除干扰`：优先排除绝对化、张冠李戴、无中生有
6. `全局复核`：检查主旨题是否和全文语气一致

## 三、题型拆解与优先级

| 题型 | 识别方式 | 做题动作 | 优先级 |
|---|---|---|---:|
| 细节题 | when/where/who/why/how | 题干关键词回文定位 | 1 |
| 词义题 | the word/phrase in line... | 看前后句逻辑与感情色彩 | 2 |
| 主旨题 | main idea / best title | 看首尾段+高频重复信息 | 3 |
| 推断题 | imply / infer / suggest | 必须“原文可推”，禁止脑补 | 4 |

## 四、定位信号词（高频）

### 1) 逻辑信号

- 转折：however, but, yet, in fact
- 因果：because, therefore, as a result
- 递进：moreover, besides, in addition
- 对比：while, unlike, by contrast

### 2) 结构信号

- 段首句：常给主题
- 转折后：常给作者真实态度
- 段尾句：常给结论/总结
- 重复词：常是文章核心线索

## 五、五类高频干扰项

| 干扰类型 | 典型表现 | 破解动作 |
|---|---|---|
| 绝对化 | some 变 all，often 变 always | 看到绝对词先警惕 |
| 张冠李戴 | A 的观点安到 B 身上 | 核对主语和代词指代 |
| 偷换因果 | 原因和结果对调 | 回文确认因果方向 |
| 以偏概全 | 局部事实扩大为整体结论 | 看是否有范围限定 |
| 无中生有 | 选项信息文中未出现 | 找不到原文依据即排除 |

## 六、时间分配（单篇参考）

- `20%`：快速通读，标段落主旨
- `55%`：细节题/词义题定位作答
- `25%`：主旨题/推断题 + 复查

> 原则：一道题卡住超过 60 秒先跳过，避免连锁丢分。

## 七、每日20分钟训练模板

做题（12分钟）：
1. 2分钟浏览题干并圈关键词
2. 8分钟完成定位题（细节+词义）
3. 2分钟处理主旨/推断题

复盘（8分钟）：
1. 记录错因：定位错 / 理解错 / 干扰项误判
2. 写出正确依据句（原文哪一句）
3. 写一句下次提醒（例如：先看转折后再选）

## 八、4周阅读提分计划

| 周次 | 目标 | 重点训练 |
|---|---|---|
| 第1周 | 提升定位速度 | 细节题专项 + 同义替换 |
| 第2周 | 稳定词义判断 | 词义题专项 + 上下文推断 |
| 第3周 | 突破主旨/推断 | 主旨题、推断题分题型训练 |
| 第4周 | 限时稳定输出 | 套题模拟 + 错题二刷 |

## 九、考前最后3天建议

1. 只做“错题回炉 + 限时小套题”，不再扩新资料
2. 每天复盘同一张“干扰项清单”
3. 保持手感优先，避免疲劳刷题

# PET3 口试全攻略 <DifficultyBadge level="B1" />

> 根据 `PETS三级口试模板.pdf` 整理，按口试三部分给出可直接套用的答题模板。本节为你提供整体考试的应对策略和各环节的详细拆解。

## 口试临场策略（简明版）

1. **不追求高级词**，优先“完整句 + 清晰逻辑”
2. **每次回答至少包含**：观点 + 原因 + 例子
3. **遇到卡壳先用过渡句占位**：`Well, in my opinion...`
4. **发音上**优先保证关键词清楚，不要抢语速

## 考试结构说明

PET3 口试主要分为三个部分，每个部分考察不同的表达能力：

- **Part A：个人信息问答（自我表达）**。考官会针对日常生活话题进行提问。
- **Part B：图片描述与互动讨论**。考官给出一组图片，要求考生描述并发表个人观点。
- **Part C：讨论表达句（主陈述 + 回应）**。要求两名考生就某一话题进行互动讨论。
